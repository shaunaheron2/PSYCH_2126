3.3.1 Descriptive Table

The first step before analysing a new dataset is to describe and visualize it. You will want to add a descriptive statistics table and a histogram of your outcome variable in your final document.

This section is meant as practice, I will list each step needed for your final paper in the “Putting it all Together” tutorial at the very end.

Luckily R has some very handy packages to help us explore our data.

Lets load the iris dataframe and store it in an object we’ll call df.

df <- data.frame(iris) # load the iris dataframe into an object we'll call df

The describe command from the dlookr packages allows you to see the mean, standard deviation and skewness of your data right away.

descriptives <- describe(df) # create an object out of the output of the describe function

3.3.2 Histograms

The next step after looking at descriptive statistics, is to visualize your data.

To see a histogram of all the continuous variables in the dataframe use the plot_histogram function from the DataExplorer package.

Histograms are a great way to check your outcome variable for normality. Remember, most statistical tests have a set of assumptions that if violated, make the results of the tests unreliable.

plot_histogram(df)

The histograms tell us a few things:

  • the Petal length histogram tells us that petal length is NOT a normal distribution. Its bimodal, meaning there are two distributions in one. Some petals are really short and some are medium to long.

  • Lucky for us our outcome variable (sepal length) seems relatively normal.

Let’s have a look at just our outcome variable. This time, we ask the function to run not just the dataframe (df) but an individual variable from the dataframe. In R, a variable name is composed of two parts:

  1. the prefix (which is the dataframe or table)
  2. the variable name (the exact name the variable has in our df)

To look at an individual variable we need to tell R where to find it. The three parts are: df (the address dataframe) $ (a connector that tells us R that we want a specific variable) Sepal.Length

Lets put it all together:

df$Sepal.Length

# To create a histogram of sepal length we call the function hist

hist(df$Sepal.Length)

Similarly, we could look at sepal width:

hist(df$Sepal.Width)

Remember: R is case sensitive, so if you dont type the names of variables and objects exactly as they appear in your dataframe, R will scream at you!

We could also look at correlations to see if there are underlying relationships we need to consider in our data:

3.3.4 T-Tests

# To run a t-test, our grouping variable can only have two levels, while Species has 4, so we;re just going to look at the difference between two levels

df2 <- df %>% filter(Species=="setosa"|Species=="versicolor") %>% droplevels()

t.test(Sepal.Length ~ Species, data = df2)
